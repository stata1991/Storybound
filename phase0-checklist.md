# Storybound — Phase 0 Checklist
## Proof of Product (Weeks 1–4)

**Goal:** Prove that Storybound's quarterly AI + illustration pipeline produces books that parents and children love — before building anything public.

**Success criteria:** 3 complete, print-ready quarterly books that a real parent would happily pay $109/year for.

---

## Week 1 — Pipeline Setup

### Story Generation
- [ ] Set up Claude API access
- [ ] Test Pass 1 (Story Bible) prompt with one child profile
- [ ] Test Pass 2 (Episode Generation) prompt using the bible output
- [ ] Refine prompts until story quality feels genuinely personalized (not templated)
- [ ] Test Episode 4 callbacks — do they feel earned or mechanical?

### Illustration
- [ ] Set up Midjourney or Replicate access
- [ ] Test face reference generation from a photo (Dreambooth/Astria)
- [ ] Test consistency across 8 scenes using same face reference
- [ ] Test across 2 age styles (early reader + chapter book)
- [ ] Evaluate: does the child actually look like the reference?

### Layout
- [ ] Set up Canva Pro or Affinity Publisher template
- [ ] Create 32-page book template (cover + 8 spreads + end pages)
- [ ] Test importing story text + illustrations into layout
- [ ] Print one test copy via Blurb or Lulu — evaluate physical quality

---

## Week 2–3 — Three Test Books

Create one complete quarterly book for each of these profiles:

### Test Child 1: The Young Adventurer
- Age: 5, girl, she/her
- Interests: dinosaurs, painting, her cat
- Avoidances: spiders, loud thunder
- Episode: Q2 (Summer) — her first adventure

### Test Child 2: The Curious Explorer
- Age: 7, boy, he/him  
- Interests: space, building things, swimming
- Avoidances: getting lost, scary monsters
- Episode: Q1 (Spring) — discovery of a hidden world

### Test Child 3: The Birthday Book Test
- Age: 9, girl, she/her
- Interests: reading, cooking, horses
- Avoidances: none specified
- Episode: Q4 (Birthday) — must include callbacks to fictional Eps 1–3

For each test book:
- [ ] Generate Story Bible
- [ ] Generate Episode text
- [ ] Generate all 9 illustrations (8 scenes + cover)
- [ ] Assemble in layout template
- [ ] Export print-ready PDF
- [ ] Order physical proof copy

---

## Week 4 — Honest Evaluation

### Internal Quality Bar
For each of the 3 books, answer honestly:

- [ ] Would you pay $109/year for this quality?
- [ ] Does the child look like the reference in the illustrations?
- [ ] Does the story feel genuinely personal or like a template with names swapped?
- [ ] Is the Episode 4 callback emotional and meaningful?
- [ ] Is the physical book quality worth a premium price?
- [ ] Would a parent show this to other parents?

### Go/No-Go Decision

**GO if:**
- All 3 books pass the quality bar above
- Physical quality is print-premium
- Face consistency is >80% across scenes
- Episode 4 birthday book produces an emotional reaction

**NO-GO if:**
- Face consistency is poor — research better tools before proceeding
- Stories feel templated — refine prompts before proceeding
- Physical quality is mediocre — find a better print partner

---

## Phase 0 Outputs

On completion, you should have:
- [ ] 3 complete print-ready quarterly books
- [ ] Refined story generation prompts (locked for Phase 1)
- [ ] Illustration style guide locked for age groups
- [ ] Print partner selected and quality verified
- [ ] Realistic COGS per book confirmed (target: <$10/book)
- [ ] Honest assessment: is this good enough to charge $109/year?
